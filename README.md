## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/fastship.git
   cd fastship
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   pnpm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   # or
   pnpm dev
   ```

4. **Build for production**
   ```bash
   npm run build
   # or
   pnpm build
   ```
